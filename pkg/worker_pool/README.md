#  WorkerPool для ограничения количества горутин

Простой воркер пул для ограничения количества горутин

```go
wp, err := New(5, 5) // создаем экземпляр WorkerPool с 5 одновременно работающими тасками
// каждый из которых может вместить до 5 задач

err = wp.Add(task) // добавление таска в очередь задач
if err != nil {
    // Если вернется ошибка ErrWorkerIsFull, значит очередь выполнения переполнена
}

wp.Close() // для закрытия WorkerPool и ожидания завершения всех поставленных в очередь задач
```
